# 자동 퀄리티 체크 + 자동 발행 로드맵

> 관련: [[2026-03-16-daily-digest-design]], [[AI-News-Pipeline-Design]]

---

## 비전

파이프라인이 생성한 뉴스 다이제스트의 퀄리티를 자동으로 평가하고, 기준 이상이면 사람 개입 없이 자동 발행.

---

## 3단계 로드맵

### Phase 1: 수동 발행 (현재)

```
파이프라인 → draft 생성 → 어드민이 확인 → 수동 Publish
```

- 모든 포스트가 draft로 생성
- 어드민이 직접 내용 확인 후 발행 여부 결정
- 퀄리티 기준 없음 (사람 판단)

### Phase 2: 퀄리티 점수 자동 평가

```
파이프라인 → draft 생성 → LLM 퀄리티 체크 → 점수 기록 → 어드민이 점수 보고 판단
```

**구현 요소:**
- `quality_check` 스테이지 추가 — 생성된 다이제스트를 별도 LLM이 평가
- 평가 기준:
  - 사실 정확성 (출처 인용 여부)
  - 섹션 완성도 (모든 카테고리 포함 여부)
  - 균형 (한쪽으로 치우치지 않았는지)
  - 가독성 (적절한 길이, 명확한 문장)
- 점수: 0~100 (또는 1~5)
- `news_posts`에 `quality_score` 필드 추가
- Pipeline Analytics의 Quality Score Trend 차트에 표시 (차트 이미 있음)
- 어드민 뉴스 리스트에서 점수 배지 표시

**비용:** gpt-4o-mini로 평가 시 ~$0.01/포스트

### Phase 3: 자동 발행

```
파이프라인 → draft 생성 → 퀄리티 체크 → 점수 ≥ 기준 → 자동 published
                                       → 점수 < 기준 → draft (수동 검수 대기)
```

**구현 요소:**
- 자동 발행 기준 점수 설정 (어드민 설정 또는 환경변수)
- 기준 이상 → status를 `published`로 자동 변경 + `published_at` 설정
- 기준 미만 → status는 `draft` 유지 + 어드민 알림 (`admin_notifications`)
- 어드민 대시보드에서 자동 발행 ON/OFF 토글
- 안전장치: 첫 N일은 "자동 발행 후보"만 표시하고 실제 발행은 안 함 (드라이런)

**기준 예시:**
- 점수 80+ → 자동 발행
- 점수 60~79 → draft + "검수 권장" 알림
- 점수 <60 → draft + "퀄리티 미달" 경고

## 비용 추적 참고

| 호출 | 비용 기록 | 비고 |
|------|---------|------|
| Tavily (뉴스 수집) | ✅ collect 스테이지 | cost_usd=$0 (무료 크레딧 사용), 호출 횟수/결과 수 추적 |
| 분류 (classify) | ✅ | gpt-4o |
| 다이제스트 생성 | ✅ | gpt-4o, 3 페르소나 × 2 타입 |
| 퀄리티 체크 | ✅ 구현 예정 | gpt-4o-mini, ~$0.01/포스트 |
| 핸드북 추출/생성 | ✅ | 별도 run |

## 선행 조건

- Phase 2 → v3 다이제스트 안정화 (DIGEST-01~04 완료)
- Phase 3 → Phase 2에서 충분한 점수 데이터 축적 + 기준 결정
