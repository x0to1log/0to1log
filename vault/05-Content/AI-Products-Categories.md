## 1. AI 어시스턴트 (`assistant`)

### 에디토리얼 설명 (KO)
모르는 게 있으면 검색 대신 물어보세요. 논문 요약, 이메일 작성, 계획 수립, 언어 번역까지 — 이제 한 명의 똑똑한 조수가 24시간 옆에 있습니다. ChatGPT, Claude, Gemini가 이 싸움을 이끌고 있습니다.

### 에디토리얼 설명 (EN)
Stop searching, start asking. Summarize papers, write emails, plan trips, translate anything — you now have a brilliant assistant available 24/7. ChatGPT, Claude, and Gemini are leading this revolution.

### 대표 제품
- ChatGPT (OpenAI)
- Claude (Anthropic)
- Gemini (Google)
- Perplexity
- Grok (xAI)
- Copilot (Microsoft)

## 2. AI 이미지 생성 (`image`)

### 에디토리얼 설명 (KO)
미술을 배운 적 없어도 됩니다. 머릿속 장면을 글로 설명하면 — 수채화, 사진, 애니메이션, 포스터 — 그 어떤 스타일로도 만들어줍니다. Midjourney 한 장이 디자이너의 하루 작업을 대체하는 시대입니다.

### 에디토리얼 설명 (EN)
No art school required. Describe the scene in your head and get back a watercolor, photograph, anime illustration, or product poster in seconds. One Midjourney prompt can replace a designer's full day.

### 대표 제품
- Midjourney
- DALL-E 3 (OpenAI)
- Stable Diffusion (Stability AI)
- Flux
- Leonardo.AI
- Adobe Firefly
- Ideogram

## 3. AI 영상 생성 (`video`)

### 에디토리얼 설명 (KO)

## 4. AI 음성/음악 (`audio`)

### 에디토리얼 설명 (KO)
악기를 다룰 줄 몰라도, 노래를 못해도 괜찮습니다. 가사를 입력하면 완성된 노래가 나오고, 텍스트를 입력하면 실제 사람 같은 목소리로 읽어줍니다. 팟캐스트, 광고, 게임 더빙이 1인으로 가능해졌습니다.

## 5. AI 코딩 도구 (`coding`)

### 에디토리얼 설명 (KO)
"코딩을 배우려면 어디서부터?"라고 묻던 시대는 끝났습니다. 원하는 기능을 설명하면 코드를 작성해주고, 버그를 찾아주고, 테스트까지 만들어줍니다. 개발자라면 생산성이 2~3배 올라갑니다.

### 에디토리얼 설명 (EN)

- Cursor
- GitHub Copilot
- Windsurf (Codeium)
- Replit AI
- v0 (Vercel, UI 생성)
- Bolt.new

## 6. AI 워크플로우 (`workflow`)

### 에디토리얼 설명 (KO)
반복되는 업무를 자동화하세요. Gmail로 들어오는 이메일을 자동으로 분류하고, Notion에 저장하고, Slack으로 알림 — 코딩 없이 가능합니다. AI가 중간 판단까지 맡아줍니다.

### 에디토리얼 설명 (EN)
Automate repetitive work. Sort incoming emails, save to Notion, notify Slack — no code needed. These tools now let AI handle the decision-making in the middle, not just the routing.

### 대표 제품
- n8n
- Make (구 Integromat)
- Zapier
- Dify (LLM 워크플로우 빌더)
- Flowise
