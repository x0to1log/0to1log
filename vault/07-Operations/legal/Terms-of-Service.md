# Terms of Service (이용약관)

## 핵심 조항

| 조항 | 내용 |
|---|---|
| **AI 생성 콘텐츠 면책** | AI 파이프라인이 생성한 뉴스/용어집은 참고 자료이며, 정확성을 보장하지 않음 |
| **서비스 중단 가능성** | Solo 프로젝트로 사전 고지 없이 서비스가 중단될 수 있음 |
| **UGC 책임** | 댓글(Phase 4 커뮤니티)의 내용은 작성자 책임 |
| **지적재산권** | 사이트 콘텐츠의 무단 복제/재배포 금지 |
| **제3자 상표 면책** | 제품명·로고·상표는 각 소유자의 자산이며, 0to1log와 제휴/보증/후원 관계 없음 |

## 향후 추가 필요 조항

| Phase | 조항 | 트리거 |
|---|---|---|
| Phase 3 후반 | DMCA 고지/삭제 절차 | 댓글/UGC 기능 추가 시 |
| Phase 3 후반 | UGC 라이선스 (사용자가 올린 콘텐츠의 사이트 사용 허락) | 댓글/UGC 기능 추가 시 |
| Phase 3 후반 | 반복 침해자 계정 정지 정책 | DMCA Safe Harbor 자격 유지 |
| Phase 4 | 결제 약관 + 환불 정책 | Premium 구독 시작 시 |
| Phase 4 | 자동 갱신 고지 + 해지 절차 | 캘리포니아 ARL 준수 |
| Phase 4 | 개인정보 제3자 제공 동의 | Stripe/Polar 결제 연동 시 |

## 구현 시점

- Phase 2: 기본 이용약관 페이지 생성 (`/en/terms/`, `/ko/terms/`)
- Phase 3: DMCA + UGC 조항 추가
- Phase 4: 구독/결제/환불 관련 조항 추가
