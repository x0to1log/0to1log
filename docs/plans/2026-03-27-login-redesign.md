# Login Page Redesign

## Goal
Redesign the existing login page to match a "Minimal & Modern" SaaS aesthetic with an asymmetric card layout, focusing on typography, whitespace, and a distraction-free user experience.

## Layout & Structure (Asymmetric Modern)
- **Background**: Soft warm-gray (`#f8f9fa` or similar).
- **Container**: A single fixed max-width card (e.g., 1024px) centered on screen with smooth `24px` rounded corners and a delicate drop shadow.
- **Split Construction (4:6)**:
  - **Left Area (40%)**: Darker or brand-color background holding the product features (Bookmarks, Reading History, Learning Progress).
  - **Right Area (60%)**: Pure white background hosting the clean login form. 
- **Responsive**: On mobile, the card switches to a vertical stack or hides the feature list entirely to prioritize the login action.

## Typography & Components
- **Typography Focus**: High contrast between headings (`Sign In`) and helper text.
- **OAuth Buttons**: Minimalist styling inspired by Linear (1px borders, subtle hover changes, monochromatic thin-stroke icons, no heavy shadows).
- **Icons**: Thin-stroke, minimalistic icons for the feature list to match the modern technical tone.
