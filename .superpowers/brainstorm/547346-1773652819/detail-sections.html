단일 컬럼에서 어떤 순서로 보여줄까요? 드래그 아닌 선택으로 — 마음에 드는 흐름을 골라주세요

A

제품 중심 (추천)

1. Hero — 로고(80px) + 이름 + 태그라인 + CTA + pricing 뱃지
2. Meta Chips — platform, korean, category (가로 칩)
3. Media Gallery — 스크린샷/영상
4. Description — 마크다운 설명
5. Key Features — 2열 카드 그리드
6. Use Cases — 배경색 카드
7. Similar Tools — 4개 카드 그리드
8. FAQ — 아코디언
9. Bottom CTA — "Visit Site" 반복

제품 이해 → 기능 확인 → 비교 → 결정 흐름

B

비주얼 먼저

1. Media Gallery — 스크린샷/영상 (풀폭, Hero 역할)
2. Hero — 로고 + 이름 + 태그라인 + CTA
3. Meta Chips — platform, pricing, korean
4. Key Features — 2열 카드 그리드
5. Description — 마크다운 설명
6. Use Cases — 배경색 카드
7. Similar Tools — 4개 카드 그리드
8. FAQ — 아코디언
9. Bottom CTA

Replicate 스타일 — 이미지가 첫인상. 하지만 이미지 없는 제품은 빈약해 보일 수 있음.
