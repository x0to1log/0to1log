상세페이지 레이아웃 방향

Sidebar를 유지할지, 단일 컬럼으로 갈지 — 클릭해서 선택해주세요

A

Clean Editorial (단일 컬럼)

Sidebar 제거. 메타 정보(pricing, platform, tags)를 Hero 아래 가로 칩으로 배치. 위에서 아래로 자연스럽게 흐름. Features는 2열 카드 그리드. 모바일과 데스크탑 경험 동일.

Pros

  • 깔끔하고 일관된 레이아웃
  • 모바일/데스크탑 같은 경험
  • 콘텐츠 흐름이 자연스러움

Cons

  • 데스크탑 여백이 넓어질 수 있음
  • Sidebar 정보가 스크롤 아래로 밀림
B

Sidebar 유지 (현재 구조 개선)

데스크탑에서 2컬럼 유지. Sidebar에 pricing, platform, tags, CTA 버튼 sticky 고정. 모바일에서는 Hero 아래 칩으로 전환.

Pros

  • 데스크탑에서 정보 밀도 높음
  • CTA가 항상 보임 (sticky)
  • 기존 코드 변경 최소화

Cons

  • 모바일에서 Sidebar가 아래로 밀림
  • 지저분해 보일 수 있음

공통 개선 (둘 다 적용)

카드에서 로고 제거 → 상세페이지에서만 로고
이모지(👁♡) → SVG 아이콘
카테고리 뱃지 → 클릭 시 /products/?cat= 링크
하단 FAQ 아래 CTA 반복
